El proyecto consta de una calculadora la cual sirve para realizar las operaciones basicas:
  - Suma
  - Resta
  - Multiplicacion
  - Division
  - Raiz cuadrada
  - Exponente al cuadrado
  - Se agrego un historial de operaciones
